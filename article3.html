Основы JavaScript

JavaScript — это язык программирования, который позволяет добавлять интерактивность на веб-страницы.

Основные возможности JavaScript

  • Манипуляция с HTML и CSS (DOM).
  • Обработка событий (клики, ввод с клавиатуры).
  • Взаимодействие с сервером (AJAX, Fetch API).
  • Работа с локальным хранилищем (localStorage, sessionStorage).
  • Создание динамического контента.

Простой пример JavaScript

document.addEventListener("DOMContentLoaded", function() {
    document.querySelector("h2").style.color = "blue";
});

Заключение

JavaScript — мощный инструмент для создания динамичных и интерактивных веб-приложений. Он является основным языком фронтенд-разработки и активно используется в современных веб-проектах.
