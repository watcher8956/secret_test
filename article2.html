Основы CSS

CSS (Cascading Style Sheets) — это язык стилей, который используется для оформления HTML-документов.

Основные свойства CSS

  • color — цвет текста.
  • background-color — цвет фона элемента.
  • font-size — размер шрифта.
  • margin — внешние отступы.
  • padding — внутренние отступы.
  • border — граница элемента.
  • display — способ отображения элемента.
  • flexbox и grid — современные технологии разметки.

Простой пример CSS

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
    margin: 20px;
}

h1 {
    color: #007bff;
}

Заключение

CSS позволяет создавать красивые и удобные сайты. Освоив его, вы сможете улучшить внешний вид страниц и сделать их более привлекательными для пользователей.
