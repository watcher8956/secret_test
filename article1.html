Основы HTML

HTML (HyperText Markup Language) — это язык разметки, используемый для создания структуры веб-страниц.

Основные теги

  • <html> — корневой элемент HTML-документа.
  • <head> — содержит метаинформацию о странице.
  • <body> — основной контент страницы.
  • <h1> - <h6> — заголовки разного уровня.
  • <p> — абзац текста.
  • <a href=\"\"> — ссылка.
  • <img src=\"\" alt=\"\"> — изображение.
  • <ul> и <li> — маркированный список.

Простой пример HTML-страницы

<!DOCTYPE html>
<html lang=\"ru\">
<head>
    <meta charset=\"UTF-8\">
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая HTML-страница.</p>
</body>
</html>

Заключение

HTML — это основа веб-разработки. Освоив его, вы сможете создавать простые веб-страницы и далее изучать CSS и JavaScript для улучшения дизайна и интерактивности.
